1. 캐릭터 구성

1-1. 캐릭터 스탯

  • 캐릭터 스탯은 아래의 테이블 입력값에 따라 설정한다.

CharacterInfo

항목자료형설명
idint항목 ID
chrNamestring캐릭터 이름, StringTable의 Key 참조
chrPathstring캐릭터 메쉬의 Path
iconPathstring캐릭터 아이콘 Path
illustPathstring캐릭터 일러스트의 Path
baseSpeedint기본 이동 속도


CharacterLevelingTable

항목자료형설명
idint항목 ID
chrIDint할당될 캐릭터 ID
참조 → 테이블: CharacterInfo - 컬럼: id
levelint캐릭터 레벨
expAmountint경험치 필요량
chrDMGint기본 공격력
chrHPint기본 체력
chrDTint데미지 감쇄값 (수치만큼 차감)
chrDRint방어력 (비율로 차감)
chrCritProbfloat기본 크리티컬 확률
chrCritDmgfloat기본 크리티컬 배율

  • 현재 기준 고유 캐릭터: 3종

  • 각 캐릭터의 스탯은 CharacterInfo에서 ID를 참조하여 CharacterLevelingTable 에 정의한다.

2-1. 캐릭터 장비

  • 각 아이콘은 현재 장착된 아이템의 아이콘 출력

    ※ 장착된 아이템이 없을 시 빈 슬롯 아이콘 출력

  • 1.5.0 에서 유물 개발 X

2-1-1. 아이템 장착 팝업

  • 장비 슬롯을 선택 시 팝업이 출력
  • 닫기 버튼 = 팝업 닫음

    A. 장비 정보

    • 선택한 슬롯의 아이템 종류에 따라 상단 정보화면의 출력 내용과 형태 달라짐.

    A-1. 무기 정보 창

    1. 등급 = #무기테이블 → $grade 참조
    1. 무기 이름 = #무기테이블→ $name
    1. 아이콘 경로 =#무기테이블→ $iconPath
    1. 착용 제한
      • [캐릭터 이름] 전용 의 형태로 출력
      • #무기테이블→ $weaponType → #CharacterInfo →$chrName
    1. 설명 및 플레버 텍스트 = #무기테이블→ $desc
    1. 레벨 = 현재 레벨 출력 (maxLevel은 따로 출력하지 않음)
    1. 기본 스탯
      • 스탯 명칭 = 공격력
      • 스탯 값 = #무기테이블→ $baseAtk + (#무기테이블→ $increaseAtk x 무기 레벨)
        • 스탯 값 출력 시, 3 자리수 단위로 콤마
          e.g. 65,535

      서브 스탯

      • 스탯 명칭 = #무기테이블→ $subStat
      • 스탯 값 = #무기테이블→ $subStatValue
    1. 아이템 잠금 버튼
      • ※ 잠금 상태일 경우 잠긴 자물쇠 출력
      • 잠금 버튼 관련 기획
      • 잠금 여부 = #유저 DB에서 호출
    1. 액션 아이콘 =

      #무기테이블→$baseAction ,$skill1Action, $skill2Action, $passiveAction
      →
      #actionGroup→$skillIconPath
      • 아이콘 터치 시 터치된 동안 액션 관련 설명 출력
      • 설명 = #무기테이블→$baseAction ,$skill1Action, $skill2Action, $passiveAction
        →
        #actionGroup →$skillName,$skillDesc
    1. 룬 슬롯
      • 선택 시 = 변경 내역 저장 후 → 룬 교체 씬 이동 ( 스킬 및 룬 태그 시스템 참고)
      • 기존 씬에서 완전 전환됨
    1. 장비 버튼
      • 현재 캐릭터에 미장착된 아이템 = 장착 출력
      • 현재 캐릭터에 장착된 아이템 = 해제 출력(무기는 해제 불가)

    B. 리스트

    1. 하단 리스트는 선택한 슬롯에 따라 필터링되어 그리드 리스트에 버튼으로 출력
      • 스크롤 방향 = 세로
      • 출력 내용 = 플레이어 보유중인 아이템 ⇒ 필터링 → 필터링 결과

      장비 항목

      • 장비 항목별 기본 요소는 다음과 같다.
      1. 등급 = #무기테이블 → $grade 참조
      1. 아이콘 =#무기테이블→ $iconPath
      1. 레벨 = 현재 레벨 출력
      1. 무기 명 = #무기테이블→ $name
      • 장착 중 = 장비 항목 전체에 딤드 처리와 함께 장착 중 표시
      • 잠금 상태 = 좌측 하단에 자물쇠 표시
        ※ 잠금 상태 X = 자물쇠 표시 X
      • 아이템 선택 시 해당 아이템 정보가 상단에 표시된다.

    2. 정렬

    • 정렬 상태에 따라 1번 리스트 정렬
    • 우선 적용 : 첫번째 아이콘 = 현재 장착중인 유물 (다른 캐릭터가 장착한 경우 포함)
    • 정렬 버튼엔 현재 정렬 상태가 출력
    • 우측 화살표 = 오름차순 내림차순 스위치 (기본값 오름차순)

2-2. 캐릭터 정보

A. 캐릭터 일러스트

B-2. 캐릭터 명

  • 선택 캐릭터가 인게임에 적용될 스탯을 표시한다.
  • 스탯 정보 창은 캐릭터의 모든 정보 표시 가능
    • 축소와 확장으로 정보량 제한
    • 기본 값 = 축소 상태
    • 축소 상태일 때 한정된 정보 출력
    • 확장 상태일 때 모든 정보 출력
    • 확장 상태일 때 스크롤 가능

  1. 스탯

    ※ 아래 링크의 공식 사용하여 인 게임 최종 스텟을 출력

    1. 공격력
    1. 체력
    1. 데미지 감소
    1. 방어력
    1. 치명타 확률 = 기본 치명타 확률 + 추가 치명타 확률
    1. 치명타 배율
    1. 이동 속도 = #CharacterInfo → $baseSpeed
